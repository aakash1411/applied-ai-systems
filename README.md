# Applied AI Systems

This repository documents the design and implementation of **production-oriented LLM systems** with a focus on reliability, determinism, and evaluation.

The goal is not to build demos, but to explore **how LLMs should be constrained and validated** in real-world, high-stakes applications.

---

## 🔍 Core Principles
- Retrieval-first architectures
- LLMs as routers, not authors
- Canonical answer units
- Deterministic outputs
- Explicit evaluation & failure handling

---

## 📦 What This Repo Contains
- Notebooks exploring intent normalization, retrieval strategies, and evaluation
- Modular Python components for routing, retrieval, and validation
- Architecture and design notes
- Companion implementations for published articles
